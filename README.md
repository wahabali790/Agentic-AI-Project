## 📦 Setup Instructions
```bash
git clone https://github.com/USERNAME/Agentic-AI-Project.git
cd Agentic-AI-Project
pip install -r requirements.txt
````

Create `.env` file:

```env
GROQ_API_KEY=your_key_here
OPENAI_API_KEY=your_key_if_any
```
